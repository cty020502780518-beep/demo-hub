技术说明

          

            本 Demo 基于 zhimeng-ai 源码结构进行静态展示。zhimeng-ai 本身已经包含 React +
            TypeScript + Vite + Ant Design + ReactFlow 的前端工作流编排层，包含 Console Hub
            后端服务，以及 Java / Python 双工作流引擎。真实工程中，前端工作流页面会生成
            nodes、edges 和 config，Console Hub 保存工作流元数据，Workflow Engine 读取 DSL
            后根据 nodeType 调度对应执行器。当前页面不调用真实后端，只用于展示我基于源码理解改造的 zhimeng-ai 工作流编排平台的产品形态和技术链路。
          

          

            画布上的节点不是 Console Hub 或 Workflow Engine 本身，而是平台预注册的业务节点模板。Console
            Hub 和 Workflow Engine 是后端平台模块；Agent、Tool、Input、Output 才是用户在画布上编排的业务节点。
