# Project Scope

本仓库是 Dream RAG 的源码展示版本，面向代码阅读与面试复盘。

## 来源与边界

- 本仓库属于学习实践、二次开发与工程化整理版本，不应对外表述为所有代码完全从零自研。
- 前端管理后台、通用组件包、工程脚手架和部分非 RAG 业务扩展可能来自开源项目或模板工程的学习与改造。
- 保留原有作者、License、NOTICE 或模板依赖声明；若后续确认上游仓库，应补充上游项目名称、链接和许可证。

## 核心阅读范围

- 文档上传、解析、切片、向量化与入库
- Kafka 异步文档处理、重试与死信队列
- Elasticsearch 关键词 + 向量混合召回
- WebSocket 流式对话与用户主动中断
- Redis 对话上下文窗口与生成状态管理

## 非核心扩展

Token 配额、充值订单、支付回调等代码是业务扩展模块，保留用于展示配额控制与商业化闭环的设计思路；阅读 RAG 主链路时可以跳过。

## 公开仓库安全说明

公开仓库仅保留 `.env.example`，真实 API Key、数据库密码、MinIO 密钥和 JWT 密钥不应提交。
