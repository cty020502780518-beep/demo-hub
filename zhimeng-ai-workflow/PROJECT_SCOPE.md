# Project Scope

本仓库是 zhimeng-ai 工作流平台的源码展示版本，面向代码阅读与面试复盘。

## 来源与边界

- 本仓库属于学习实践、二次开发与工程化整理版本，不应对外表述为所有代码完全从零自研。
- Console、后台管理、工程脚手架、通用工具类和部分示例数据可能来自开源项目或模板工程的学习与改造。
- 保留原有作者、License、NOTICE 或模板依赖声明；若后续确认上游仓库，应补充上游项目名称、链接和许可证。

## 核心阅读范围

- ReactFlow 可视化编排与工作流 DSL
- Java WorkflowEngine、NodeExecutor 与边驱动 DAG 调度
- SSE 实时事件推送与流式输出
- LLM / 插件节点执行链路
- Docker Compose 本地开发环境

## 公开仓库安全说明

公开仓库中的环境变量均为示例值，真实 API Key、对象存储密钥、数据库密码和 JWT 密钥不应提交。
